# A-Treasure-Hunt-through-NYC-Open-Data
##Repo for attendees of NYC Open Data Week Event "A Treasure Hunt through NYC Open Data" on March 23, 2026. 

Users may wish to download these datasets or to copy the Google Colab notebook into their own drive and work there. 

Link to Colab Notebook https://colab.research.google.com/drive/1p543PLgdlfcSiQaWuPDUp3bx1nwtcK3Y?usp=sharing

Large datasets have been pulled (possibly filtered) from NYC Open Data via API using Google Colab Notebook, small ones were downloaded directly from the NYC Open Data website.

Changes to full datasets:

- Bicycle Counts for East River Bridges (Historical) - Trace Replaced.csv  combines datasets from different months and replaces trace precipitation (denoted "T") with 0 precipitation for the purposes of this demonstration. 

- Motor Vehicle Collisions - Crashes_2023-2024.csv limits crashes to 2023 - 2024, not the full time period

- 2023YellowTaxiTrip_First100k.csvonly looks at the first 100k trips where the fare > 0

- 311 Service Requests from 2020 to Present_COOPER_SEAMAN.csv limits requests to those regarding these 2 streets
